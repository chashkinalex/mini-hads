# User Flows

## Doctor

1. Open mini app in Telegram, MAX, or VK
2. Pass platform authentication
3. Tap `Create survey`
4. Show QR to patient
5. Wait for status changes
6. Receive realtime result

## Patient

1. Scan QR
2. Open the join link
3. Read the intro
4. Answer 14 HADS questions
5. Submit the form
6. See a completion screen
